# Exemple d'erreur de build de react-dsfr
Ce projet permet de reproduire le ticket [#484](https://github.com/codegouvfr/react-dsfr/issues/484) de [react-dsfr](https://github.com/codegouvfr/react-dsfr). 

## Comment l'utiliser
1. Cloner ce repo
2. Installer ses dépendances avec `npm install`
3. Essayer de build : `npm run build`, ce qui échoue
